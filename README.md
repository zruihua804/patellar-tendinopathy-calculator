# 髌腱病评估、康复分层与随访计算器

面向医生与康复治疗师的髌腱病评估工具：授权中文版 VISA-P、VAS、下肢 ROM 滑条、康复记录、自动随访趋势、患者摘要、病历文本与飞书同步。

## 临床边界

- 当前模型版本为 `PT-v0.3-wide-rom-followup-2026-07-15`：输出 24 周重返原先运动水平的文献沟通参考（规律完成结构化康复与未规律完成结构化康复两条路径），不输出手术概率或自动治疗决定。该参考锚定于渐进肌腱负荷研究总体结果，并以当前功能、疼痛、病程和近期康复执行度作小幅调整；不是外部验证的个体预测模型。
- VISA-P 中文题干与计分基于临床用户于 2026-07-14 提供并确认可用的授权文件；公开发布或授权范围变更前须再次确认。
- 截图 OCR 仅在应用进程内完成，原始图像不写入本地 CSV、飞书或日志；识别结果必须人工确认。
- ROM 由患侧/健侧膝关节、患侧髋关节和患侧踝膝靠墙的滑条完成；膝靠墙按标准以距离（cm）记录。每次评估只保存一条 ROM 综合记录，所有数值各占一个列；每次保存后，VISA-P 与指定负荷 VAS 会自动写入随访趋势。

## 本地运行

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

运行测试：

```bash
python3 -m unittest discover -s tests -v
```

当前原型自动测试覆盖完整 VISA-P 计分、截图信息解析、ROM/VAS 字段映射、飞书文本数值的趋势计算、文献沟通参考，以及患者 ID 驱动的合成病例保存与更新；未使用或保留任何真实患者信息。

## 飞书（第二授权闸门）

1. 在临床用户或机构账号下创建飞书多维表格，并将应用加为协作者。
2. 在飞书开放平台申请权限、发布应用。
3. 将 `.streamlit/secrets.toml.example` 复制为 `.streamlit/secrets.toml`，填写团队访问口令、App ID、App Secret 和用户所有的 Base/Wiki 链接。
4. 飞书界面保留四张工作表：患者主表、髌腱病评估表、ROM 综合评估、患者随访总览。所有业务表只显示患者 ID；评估及 ROM 以“患者 ID + 时间点 + 日期”在后台安全更新，不显示额外 ID。
