# 髌腱病评估、康复分层与随访计算器

本地 Streamlit 原型，面向医生与康复治疗师的髌腱病评估、授权中文版 VISA-P、ROM/康复记录、患者解释、病历文本、稳定 ID 保存和飞书同步准备。

## 临床边界

- 当前模型版本为 `PT-v0.1-trend-only-2026-07-14`：记录和解释趋势，不输出个体恢复概率、手术概率或自动治疗决定。
- VISA-P 中文题干与计分基于临床用户于 2026-07-14 提供并确认可用的授权文件；公开发布或授权范围变更前须再次确认。
- 截图 OCR 仅在应用进程内完成，原始图像不写入本地 CSV、飞书或日志；识别结果必须人工确认。
- 云端语音仅用于康复师主动按住按钮时的短句 ROM 录入。浏览器调用其云端语音转写能力；应用不连续监听、不保存音频，最终文字仅暂存到康复师确认或丢弃为止。请使用支持网页语音转写的最新版 Chrome 或 Edge；任何失败均可手工录入。

## 本地运行

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

运行测试：

```bash
python3 -m unittest discover -s tests -v
```

当前原型已通过 21 项自动测试，包括完整 VISA-P 计分、截图信息解析、云端语音短句结构化解析、跨七张记录表的合成病例保存与稳定 ID 更新；未使用或保留任何真实患者信息。

## 飞书（第二授权闸门）

1. 在临床用户或机构账号下创建飞书多维表格，并将应用加为协作者。
2. 在飞书开放平台申请权限、发布应用。
3. 将 `.streamlit/secrets.toml.example` 复制为 `.streamlit/secrets.toml`，填写团队访问口令、App ID、App Secret 和用户所有的 Base/Wiki 链接。
4. 应用只会新增缺失的表/字段；以稳定临床 ID 更新；发现重复记录时停止并提示人工核查，不自动删除历史数据。
